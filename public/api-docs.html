Velo API Reference

All endpoints are protected by an x-api-key header. No Bearer tokens are required for the current local flow.
Base URL: http://localhost:3008

Base URL: http://localhost:3008/api/v1

Authentication

POST
/auth/request-otp
Send a one-time password to a phone number

Headers

Key Value Required
x-api-key 12345 ✅
Content-Type application/json ✅

Request Body

Field Type Description
phoneNumberrequired string E.164 format, e.g. +233500000000

Example

{
  "phoneNumber": "+233500000000"
}

Response Codes

200 — OTP sent (check server logs) 400 — Phone number missing 403 — Invalid API key 500 — Server error
POST
/auth/verify-otp
Verify OTP and get the user's ID

Headers

Key Value Required
x-api-key 12345 ✅
Content-Type application/json ✅

Request Body

Field Type Description
phoneNumberrequired string The phone number the OTP was sent to
coderequired string 6-digit OTP from server logs

Example

{
  "phoneNumber": "+233500000000",
  "code": "819423"
}

Success Response

{
  "token": "mock-jwt-token",
  "user": {
    "id": "a060d82d-ef33-4647-8501-89403be34948",
    "is_new_user": true,
    "roles": []
  }
}

Response Codes

200 — Verified, returns token + user 401 — Invalid or expired OTP 403 — Invalid API key

Profile Setup

POST
/profile/buyer
Create or update a buyer profile for a user

Headers

Key Value Required
x-api-key 12345 ✅
Content-Type application/json ✅

Request Body

Field Type Description
userIdrequired uuid From verify-otp response
fullNamerequired string Full display name
emailoptional string Email address
regionoptional string Region, e.g. Greater Accra
primaryLocationoptional string Specific area, e.g. Osu

Example

{
  "userId": "a060d82d-ef33-4647-8501-89403be34948",
  "fullName": "John Doe",
  "email": "john@example.com",
  "region": "Greater Accra",
  "primaryLocation": "Osu"
}

Response Codes

201 — Profile created 400 — Missing required fields 403 — Invalid API key 500 — Server error
POST
/profile/driver
Create or update a driver profile, including Ghana Card & vehicle info

Headers

Key Value Required
x-api-key 12345 ✅
Content-Type application/json ✅

Request Body Fields

Field Type Description
userIdrequired uuid From verify-otp response
fullNamerequired string Driver's full name
idNumberrequired string Ghana Card number, e.g. GHA-123456789-0. Must be unique.
vehicleTyperequired string e.g. Motorcycle, Car, Bicycle
plateNumberrequired string Vehicle plate, e.g. GT-1234-24
licenseNumberrequired string Driver's license number
licensePhotoUrlrequired string (url) Public URL of the license photo
idImageUrlrequired string (url) Public URL of the Ghana Card photo

Example Request

{
  "userId": "a060d82d-ef33-4647-8501-89403be34948",
  "fullName": "Jane Doe",
  "idNumber": "GHA-987654321-0",
  "vehicleType": "Motorcycle",
  "plateNumber": "GT-999-24",
  "licenseNumber": "DL-2024-GH-00123",
  "licensePhotoUrl": "https://cdn.example.com/license.jpg",
  "idImageUrl": "https://cdn.example.com/ghana-card.jpg"
}

Success Response

{
  "id": "b2c8e1a0-4f77-4abc-9301-1234abcd5678",
  "userId": "a060d82d-ef33-4647-8501-89403be34948",
  "fullName": "Jane Doe",
  "vehicleType": "Motorcycle",
  "plateNumber": "GT-999-24",
  "licenseNumber": "DL-2024-GH-00123",
  "status": "pending",
  "identificationId": "xyz-uuid",
  "createdAt": "2026-03-01T08:00:00.000Z"
}

⚠️ Notes

  • The idNumber must be unique across all identifications. Reusing an existing Ghana Card number will return a 500 error.
  • Status is always pending on creation. An admin must approve to change to approved.
  • The driver role is automatically assigned upon successful creation.

Response Codes

201 — Profile created 400 — Missing required fields 403 — Invalid API key 500 — Duplicate ID number or server error
POST
/profile/merchant
Create or update a merchant (business) profile

Headers

Key Value Required
x-api-key 12345 ✅
Content-Type application/json ✅

Request Body Fields

Field Type Description
userIdrequired uuid From verify-otp response
businessNamerequired string Registered business name
categoryrequired string e.g. Food & Beverage, Electronics
addressrequired string Business physical address
idNumberrequired string Ghana Card number (must be unique)
idImageUrlrequired string (url) Public URL of the Ghana Card photo
businessEmailoptional string Business contact email
businessPhoneoptional string Business contact phone
latitudeoptional number GPS latitude
longitudeoptional number GPS longitude
registrationDocUrloptional string (url) Business registration document URL

Example

{
  "userId": "a060d82d-ef33-4647-8501-89403be34948",
  "businessName": "Velo Cuisine",
  "category": "Food & Beverage",
  "address": "Osu, Accra",
  "idNumber": "GHA-111222333-0",
  "idImageUrl": "https://cdn.example.com/id.jpg",
  "businessEmail": "hello@velocuisine.com",
  "businessPhone": "+233200000000",
  "latitude": 5.556,
  "longitude": -0.196
}

Response Codes

201 — Profile created 400 — Missing required fields 403 — Invalid API key 500 — Duplicate ID number or server error
