API Documentation

Welcome to the Velo Backend API documentation. This API powers the Velo ride-hailing and courier platform, providing robust services for users, drivers, and merchants.

Base URL: /api/v1 Auth: x-api-key header

Authentication

POST /auth/request-otp
API Key Required

Requests a 6-digit verification code to be sent via SMS/WhatsApp to the provided phone number.

Request Body 
{
  "phoneNumber": "+233536690447"
}
Response (200 OK) 
{
  "message": "OTP sent successfully"
}
POST /auth/verify-otp
API Key Required

Verifies the OTP code received by the user and initiates the synchronization process.

Request Body 
{
  "phoneNumber": "+233536690447",
  "code": "123456"
}
Response (200 OK) 
{
  "message": "OTP verified",
  "synced": true
}

Profiles

POST /profile/buyer
Buyer Only API Key Required

Sets up or updates the buyer profile for the authenticated user.

Request Body 
{
  "fullName": "John Doe",
  "region": "Greater Accra",
  "primaryLocation": "Adabraka, Accra"
}
Response (201 Created) 
{
  "id": "uuid-string",
  "fullName": "John Doe",
  "status": "active"
}

Rides

POST /rides/estimate
Buyer Only API Key Required

Gets fare estimates for all available vehicle types based on pickup and dropoff locations.

Request Body 
{
  "origin": { "lat": 5.6037, "lng": -0.1870 },
  "destination": { "lat": 5.6148, "lng": -0.2058 }
}
Response (200 OK) 
[
  {
    "vehicleType": "bike",
    "estimatedFare": 15.50,
    "distance": "3.2 km",
    "duration": "8 mins"
  },
  {
    "vehicleType": "standard",
    "estimatedFare": 35.00,
    "distance": "3.2 km",
    "duration": "12 mins"
  }
]
POST /rides/request
Buyer Only API Key Required

Requests a new ride. The system will look for nearest online drivers.

Request Body 
{
  "origin": { "lat": 5.6037, "lng": -0.1870, "address": "Airport" },
  "destination": { "lat": 5.6148, "lng": -0.2058, "address": "Osu" },
  "vehicleType": "standard",
  "paymentMethod": "wallet"
}
Response (201 Created) 
{
  "rideId": "uuid-string",
  "status": "searching",
  "message": "Searching for nearby drivers"
}

Driver Operations

POST /driver/online
Driver Only API Key Required

Sets the driver's status to online, making them eligible to receive ride requests.

Request 
{}
Response (200 OK) 
{
  "status": "online",
  "message": "You are now online"
}
POST /driver/location
Driver Only API Key Required

Updates the driver's current real-time GPS coordinates.

Request Body 
{
  "lat": 5.6037,
  "lng": -0.1870,
  "heading": 180
}
Response (200 OK) 
{ "success": true }
